Capabilities

Built for code and terminals.

Every glyph is exactly one cell. Clean lines, unambiguous characters, and fully featured for development and writing.

Unambiguous Glyphs

Features a slashed zero, footed one, tailed ell, and serifed cap-I so hashes and IPs are perfectly readable at 2 a.m.

Coding Ligatures

Beautifully styled operator ligatures for ->, !=, ==, ===, <=, >=, :=, and :: via standard calt features.

Terminal Box-Drawing

Perfectly aligned box-drawing elements (light/heavy/double/rounded), block elements, shades, and Powerline separators.

Six Metric-True Styles

Regular, Medium, and Bold — plus matching true-italic obliques. Every glyph is exactly 600 UPM wide across all weights.

Workflow

Install anywhere.

1

Desktop TTF

Install the TTF files on Windows, macOS, or Linux using standard tools for use in terminals, editors, and apps.

2

Obsidian CSS Snippet

Drop the provided shellback-mono.css snippet into your vault to instantly theme your entire Obsidian workspace with embedded fonts.

3

Web WOFF2

Deploy the web font payload (~5 KB per style) to your static site or web app for blazingly fast typographic delivery.

Get Started

Install in under a minute.

Recommended

BRAT

  1. Install the BRAT community plugin.
  2. Open BRAT settings and click Add Beta plugin.
  3. Paste this repository URL.
  4. Enable Shellback Mono in your plugin list.

Manual

  1. Download main.js, manifest.json, and styles.css from the latest release.
  2. Create .Shellback Mono/plugins/{{REPO_NAME_LOWER}}/ inside your vault.
  3. Place the three files inside that folder.
  4. Reload Shellback Mono and enable the plugin.
