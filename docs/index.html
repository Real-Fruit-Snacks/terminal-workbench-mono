Terminal Workbench Mono

The official monospace of the Terminal Workbench design system — clean geometric letterforms, a slashed zero, unambiguous 0O Il1 5S 8B, eleven coding ligatures and a full terminal pack. Six styles, generated from a parametric Python engine, OFL licensed.

Code
workbench.pyutf-8
# attach every pane in the session
def attach(session, count=4):
    panes = [Pane(i) for i in range(count)]
    if session.status == "active":
        print(f"attached -> {panes}")
    return panes
Install
Desktop · TTF Unzip TerminalWorkbenchMono-TTF.zip, install all six TTFs, then pick "Terminal Workbench Mono" in your terminal or editor.
Obsidian · Snippet Drop terminal-workbench-mono.css into .obsidian/snippets/ and enable it — fonts embedded, nothing to install.
Web · WOFF2 Link terminal-workbench-mono.css next to a fonts/ folder — six styles, ~5 KB each.
